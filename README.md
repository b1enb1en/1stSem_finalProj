# ReserbaSilid

**ReserbaSilid** A web-based room reservation and scheduling management system designed for schools or institutions. It allows users to view real-time room availability, manage class schedules, and book rooms for specific time slots.

## Technologies:
- Back-End: PHP (Native), SQLite (Database)
- Front-End: HTML5, CSS3, JavaScript (embedded)
- Server: PHP Built-in Server (for development)

## Features:
- School recommendations
- Comprehensive Comparisons  
- Accurate and Updated Information
- Student-Centered Design
- Affordable and Specialized Search Options
